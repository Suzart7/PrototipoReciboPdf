# Protótipo — Gerador de Documentos

Uma ferramenta simples para criar **orçamentos, recibos e pedidos** em poucos segundos, direto do celular ou do computador, sem precisar instalar nada.

## Acesso

🔗 **https://suzart7.github.io/PrototipoReciboPdf/**

Funciona em qualquer navegador — Chrome, Safari, etc. Não precisa de internet depois que a página carrega pela primeira vez.

## Como usar

1. Escolha o tipo de documento: **Orçamento**, **Recibo** ou **Pedido**.
2. Preencha os dados de quem emite, do cliente e os itens (pode adicionar quantos quiser — o total soma sozinho).
3. Clique em **Gerar** para ver o documento pronto.
4. Use **Imprimir / Salvar PDF** para baixar o arquivo final.

O sistema avisa na hora se algum campo obrigatório estiver faltando ou incorreto.

## Sobre esta versão

Este é um **protótipo de demonstração** — o objetivo é validar o fluxo, a parte visual fazeremos depois para chegarmos na versão final. Por isso, os documentos gerados aqui saem com uma marca d'água e não devem ser usados oficialmente.

A versão definitiva remove a marca d'água e pode incluir, conforme a necessidade: numeração sequencial automática, histórico de documentos emitidos e cadastro de clientes.
